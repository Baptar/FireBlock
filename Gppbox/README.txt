Projet : Enjminshake

Auteur : Baptiste Ruberte

Description : travail de gameplay programming avec pour objectif de rendre le jeu fluide, avec un bon gamefeel

Etat avancement : j'ai terminé toutes les tâches que je souhaitais faire, je n'ai pas fait celle du "Pet Drone" (car on peut sauter une tâche)

commandes :
CLAVIER SOURIS
- Q 	      : move left
- D 	      : move right
- space       : jump
- mouse left  : death ray (ou ajouter élément dans l'éditeur de niveau)
- mouse right : fire (ou retirer élément dans l'éditeur de niveau)
- E 	      : homing magic missile
- R 	      : reload

MANETTE
- joystick gauche : move right/left
- button bottom   : jump
- L2		  : death ray
- R2		  : fire
- button right    : homing magic missile
- button left     : reload
